== Frequently Asked Questions ==
= How many menu does Idyllic theme support? =
There is four Menu registered in this theme. i.e. Main Menu,  Side Menu,  Footer Menu  and Add Social Icons Only

= Where can i get instruction about theme? = 
For detail instructions visit https://themefreesia.com/theme-instruction/idyllic/

= Is theme compatible with wpml? =
wpml-config.xml is used to support multilangual plugins such and wpml, polylang.

== Installation ==
1. Primary: == Installation ==
	- In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
	- Type in Idyllic in the search form and press the 'Enter' key on your keyboard.
	- Click on the 'Activate' button to use your new theme right away.
	- Go to https://themefreesia.com/theme-instruction/idyllic/ for a guide on how to customize this theme.
	- Navigate to Appearance > Customize in your admin panel and customize to taste.

2. Alternate: Unzip the template file (idyllic.zip) that you have downloaded. Via FTP, upload the whole folder (idyllic) to your server and place it inside the http://yourdomain.com/wp-content/themes/folder(idyllic). Do not change directory name. The template files should thus be here now: http://yourdomain.com/wp-content/themes/idyllic/index.php (for example).
	- Log into your WP admin panel and click on the Appearance -> Theme tab. Now click on the Idyllic theme to activate it.
	- Want to enjoy Idyllic feature. Go to  Dashboard -> Appearance -> Customize and play with Idyllic Options
	- The theme is offered free of charge. If someone asked money for it, someone just tricked you.
