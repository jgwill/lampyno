== Description ==

The Materialis Companion plugin adds drag and drop page builder functionality to the Materialis theme.

The Materialis Companion plugin features include:

- Beautiful ready-made homepage
- Drag and drop page customization
- 30+ predefined content sections
- Live content editing
- and many other features
